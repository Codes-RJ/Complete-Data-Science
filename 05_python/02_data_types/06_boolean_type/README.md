# Boolean Type (`bool`) ✅❌

---

## What is `bool`?

`bool` has only two values:
- `True`
- `False`

Used in:
- conditions (`if`, `while`)
- comparisons (`==`, `>`, `<`)
- logical operations (`and`, `or`, `not`)

---

## Truthy vs Falsy

In Python, many values behave like True/False in conditions.

### Common falsy values
- `False`
- `0`, `0.0`
- `""` (empty string)
- `[]`, `{}`, `set()` (empty containers)
- `None`

Everything else is usually truthy.
