# Type Conversion 🔄

---

## What is Type Conversion?

Type conversion means converting one data type into another.

Two categories:
- **Explicit conversion**: you convert manually using functions like `int()`, `float()`, `str()`, `list()`, `set()`, `dict()`
- **Implicit conversion**: Python converts automatically in some operations (mostly numeric)

## Key Rule (Python is strongly typed)

Python will not automatically mix unrelated types:

```python
"5" + 3  # TypeError
```
