# Set Types 🧺

---

## What are Sets?

Sets are collections of **unique** items.

Two set types in Python:
- **`set`** → mutable (can add/remove)
- **`frozenset`** → immutable (cannot change)

---

## Key Features

- **No duplicates**
- **Unordered** (no index-based access)
- **Fast membership checks** using `in`
- Support **math-like operations**:
  - union `|`
  - intersection `&`
  - difference `-`
  - symmetric difference `^`
