# Binary Types 🧪

---

## What are Binary Types?

Binary types represent **raw bytes** (0–255), commonly used in:
- file I/O
- images/audio/video data
- networking (packets)
- encryption/hashing

Python provides:
- **`bytes`** → immutable bytes
- **`bytearray`** → mutable bytes
- **`memoryview`** → view into existing bytes-like data (no copy)
