# Text Type (`str`) 🧵

---

## What is `str`?

`str` is Python’s **text type**:
- **Unicode** (can store characters from any language)
- **immutable** (cannot change in-place)
- behaves like a **sequence** (supports indexing and slicing)

---

## Quick Examples

```python
s = "Python"
print(s[0])    # P
print(s[-1])   # n
print(s[1:4])  # yth
```
