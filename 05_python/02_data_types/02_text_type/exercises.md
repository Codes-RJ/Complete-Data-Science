# Exercises — Text Type (`str`) 📝

---

## A. Basics

1. Create a string in three ways:
   - double quotes
   - single quotes
   - triple quotes (multi-line)
2. Given `s = "data science"`, print:
   - `s.upper()`
   - `s.title()`
   - first 4 characters
   - last 4 characters
   - reversed string
3. Demonstrate immutability by trying to change a character in a string. Show the correct way.

---

## B. Methods Practice

Given:

```python
text = "  Python is FUN!!  "
```

1. Remove extra spaces.
2. Convert to lowercase.
3. Replace `"FUN"` with `"POWERFUL"`.
4. Count how many times `"n"` appears (case-sensitive vs case-insensitive).

---

## C. Splitting and parsing

1. Given `email = "user@example.com"`, split into:
   - username
   - domain
2. Given `csv_line = "Ava,20,91.5"`, split and store in variables:
   - name (str)
   - age (int)
   - marks (float)

---

## D. Formatting

1. Ask the user for a name and print a greeting using an f-string.
2. Format the number `12345.6789` as:
   - 2 decimals
   - comma separated with 2 decimals
   - scientific notation (3 digits)
3. Print a neat table of 1 to 5 with their squares:

Example output:
```
 n | n^2
---------
 1 | 1
 2 | 4
 3 | 9
 4 | 16
 5 | 25
```

Write a program that:
