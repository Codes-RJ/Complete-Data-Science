# Mapping Type (`dict`) 🗂️

---

## What is a Dictionary?

A **dictionary** stores **key → value** pairs.

Key points:
- keys must be **hashable** (immutable types like `str`, `int`, `tuple` of immutables)
- values can be **any type**
- lookup by key is usually **fast** (average \(O(1)\))
- dictionaries preserve **insertion order** (Python 3.7+)
