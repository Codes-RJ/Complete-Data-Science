# Detailed Learning Path 📖

Complete breakdown of topics to master in sequence.

---

## PHASE 1: FOUNDATIONS

### Week 1-2: Introduction to Data Science
- What is Data Science?
- Data Science Lifecycle
- Applications across industries
- Tools and technologies overview
- Ethics in Data Science
- Key terminology

### Week 3: Excel for Data Analysis
- Excel basics and navigation
- Formulas and functions (VLOOKUP, INDEX-MATCH)
- Data cleaning in Excel
- Pivot tables
- Creating dashboards
- Macros and VBA basics

### Week 4-5: Python/R Programming
**Python Track:**
- Variables and data types
- Operators and expressions
- Control statements (if-else, loops)
- Functions and modules
- File handling
- Error handling
- OOP basics

**OR**

**R Track:**
- Data types and structures
- Vectors and data frames
- Functions
- dplyr for data manipulation
- ggplot2 for visualization

### Week 6-7: Statistics & Probability
- Descriptive statistics (mean, median, mode, variance, standard deviation)
- Probability basics
- Probability distributions (normal, binomial, Poisson)
- Sampling techniques
- Hypothesis testing (t-test, chi-square)
- Confidence intervals
- Correlation and causation
- Bayesian statistics basics

---

## PHASE 2: DATA ANALYSIS

### Week 8: SQL for Data Science
- Basic queries (SELECT, WHERE, ORDER BY)
- Filtering and sorting
- Joins (INNER, LEFT, RIGHT, FULL)
- Aggregations and GROUP BY
- Subqueries and CTEs
- Window functions
- Query optimization basics

### Week 9-10: Pandas & NumPy
**NumPy:**
- Arrays and array operations
- Indexing and slicing
- Broadcasting
- Linear algebra operations
- Random sampling

**Pandas:**
- Series and DataFrames
- Data import/export
- Data cleaning (handling nulls, duplicates)
- Data manipulation (filtering, sorting)
- GroupBy operations
- Merge, join, concatenate
- Time series data
- Pivot tables

### Week 11: Data Visualization
**Matplotlib:**
- Line plots, bar charts, histograms
- Scatter plots
- Customization and styling
- Subplots
- 3D plots

**Seaborn:**
- Distribution plots
- Categorical plots
- Heatmaps
- Regression plots
- Grids and styling

**BI Tools (Choose one):**
- **Power BI:** Data import, transformation, DAX, dashboards
- **Tableau:** Data connection, visualizations, dashboards, storytelling

---

## PHASE 3: MACHINE LEARNING

### Week 12: ML Fundamentals
- What is Machine Learning?
- Supervised vs Unsupervised Learning
- Train-test split
- Feature scaling
- Handling categorical data
- Bias-variance tradeoff

### Week 13: Supervised Learning - Regression
- Linear Regression
- Polynomial Regression
- Ridge and Lasso Regression
- Evaluation metrics (MAE, MSE, RMSE, R²)

### Week 14: Supervised Learning - Classification
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Evaluation metrics (accuracy, precision, recall, F1, ROC-AUC)

### Week 15: Unsupervised Learning
- Clustering (K-Means, Hierarchical, DBSCAN)
- Principal Component Analysis (PCA)
- Dimensionality reduction

### Week 16: Model Optimization
- Hyperparameter tuning (GridSearchCV, RandomSearchCV)
- Cross-validation
- Ensemble methods (Bagging, Boosting)
- XGBoost, LightGBM, CatBoost

### Week 20: Generative AI & LLMs

## 🎓 Prerequisites Before Starting

- Basic mathematics (algebra, calculus basics)
- Logical thinking
- A computer with internet connection

## 📝 Tips for Success

1. **Code daily** - Even 30 minutes helps
2. **Document your learning** - Create notes
3. **Build projects** - Apply what you learn
4. **Join communities** - Learn from others
5. **Don't compare** - Everyone learns at their own pace
