# Assets 📁

This folder contains visual assets used throughout the repository.

## Folder Structure

```
assets/
├── images/          # Screenshots, illustrations, logos
├── diagrams/        # Architecture diagrams, flowcharts
└── gifs/            # Animated demonstrations
```

## Usage Guidelines

### Images
- Format: PNG or JPEG
- Resolution: Minimum 1200px width for full-page images
- Naming: `topic_description.png` (e.g., `pandas_dataframe_structure.png`)

### Diagrams
- Format: PNG, SVG, or draw.io source files
- Include source files when possible for easy editing
- Naming: `diagram_name.drawio` and `diagram_name.png`

### GIFs
- Format: GIF or MP4
- Max size: 10MB
- Naming: `action_demonstration.gif`

## How to Reference Assets

In markdown files:
```markdown
![Alt text](../assets/images/image_name.png)

![Data Science Lifecycle](../assets/diagrams/ds_lifecycle.png)

![Demo](../assets/gifs/demo.gif)
```
