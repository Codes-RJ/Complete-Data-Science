# Key Concepts and Terminology 📖

## Overview

Every field has its own language. This file defines the essential terms you'll encounter throughout your data science journey.

---

## Core Concepts

### Data
Raw facts, figures, and observations
- **Structured data:** Organized in rows and columns (Excel, SQL tables)
- **Unstructured data:** Text, images, videos, audio
- **Semi-structured data:** JSON, XML (has some organization)

### Information
Data that has been processed, organized, and given meaning

### Insight
Actionable understanding derived from information

### Knowledge
Accumulated insights that enable better decision-making

---

## Types of Data

### Numerical
- **Continuous:** Any value (height, weight, temperature)
- **Discrete:** Countable integers (number of customers, items sold)

### Categorical
- **Nominal:** No order (colors, cities, product types)
- **Ordinal:** Has order (ratings: low, medium, high)

### Text
Unstructured written content (reviews, tweets, documents)

### Time Series
Data points indexed by time (stock prices, sales by month)

### Image/Video
Visual data used in computer vision

---

## Data Science Terminology

### Dataset
A collection of related data

### Features (Variables)
The input columns/attributes used for analysis
- **Independent variables:** Features used to make predictions
- **Dependent variable (target):** What we're trying to predict

### Observations (Records, Rows)
Individual data points or entries

### Training Data
Data used to teach a model

### Testing Data
Data used to evaluate a model's performance

---

## Statistics Terms

### Mean
Average of all values

### Median
Middle value when data is sorted

### Mode
Most frequently occurring value

### Standard Deviation
How spread out values are from the mean

### Variance
Square of standard deviation

### Correlation
How two variables move together (ranges from -1 to 1)

### Distribution
How values are spread across possible values

### Outlier
Value significantly different from others

### Population
Entire group you want to study

### Sample
Subset of the population used for analysis

---

## Machine Learning Terms

### Model
Algorithm that learns patterns from data

### Algorithm
Step-by-step procedure for solving a problem

### Training
Process where model learns from data

### Prediction
Output generated by a trained model

### Feature Engineering
Creating new features from existing data

### Overfitting
Model learns noise instead of patterns (too complex)

### Underfitting
Model fails to capture patterns (too simple)

### Bias
Error from wrong assumptions

### Variance
Error from sensitivity to training data

### Bias-Variance Tradeoff
Balancing model complexity for optimal performance

### Hyperparameters
Settings configured before training (not learned from data)

### Parameters
Values learned from training data

---

## Supervised Learning Terms

### Label
The target value we're trying to predict

### Classification
Predicting a category (spam or not spam)

### Regression
Predicting a number (house price)

### Binary Classification
Two categories (yes/no)

### Multi-class Classification
More than two categories (dog, cat, bird)

---

## Unsupervised Learning Terms

### Clustering
Grouping similar data points

### Dimensionality Reduction
Reducing number of features while preserving information

### Principal Component Analysis (PCA)
Common dimensionality reduction technique

---

## Model Evaluation Terms

### Accuracy
Percentage of correct predictions

### Precision
Of positive predictions, how many were correct

### Recall
Of actual positives, how many were identified

### F1 Score
Harmonic mean of precision and recall

### Confusion Matrix
Table showing correct and incorrect predictions

### ROC Curve
Plot of model performance across thresholds

### AUC
Area under ROC curve (higher = better)

### MSE (Mean Squared Error)
Average squared difference between predictions and actual

### MAE (Mean Absolute Error)
Average absolute difference between predictions and actual

### R²
How well model explains variance in data

---

## Deep Learning Terms

### Neural Network
Model inspired by the human brain

### Neuron (Node)
Basic unit in a neural network

### Layer
Collection of neurons

### Input Layer
Receives features

### Hidden Layer
Intermediate layers that learn patterns

### Output Layer
Produces predictions

### Activation Function
Decides if neuron should "fire"

### Weights
Strengths of connections between neurons

### Backpropagation
Algorithm for training neural networks

### Epoch
One complete pass through training data

### Batch
Subset of data processed together

### CNN (Convolutional Neural Network)
For image data

### RNN (Recurrent Neural Network)
For sequential data

### Transformer
Modern architecture for text (GPT, BERT)

---

### Velocity

### Variety

### Veracity

### Value

### Data Lake
